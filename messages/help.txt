Для решения задач можно использовать одну из следующих комманд:

/gcd - наибольший общий делитель
/lineareq - линейное уровнение по модулю
/poleq - полиномиальное уравнение по модулю (пока что не работает)
/jacobi - символ якоби
/rootmod - квадратичное уравнение по модулю простого числа

Также можно отменить текущее действие коммандой /cancel
