*Для решения задач можно использовать одну из следующих комманд:*

/gcd - наибольший общий делитель
/lineareq - линейное уравнение по модулю
/jacobi - символ якоби
/rootmod - квадратичное уравнение по модулю простого числа

*Также есть такие команды:*

/cancel - отменить текущее действие
/help - получить инфу по командам (этот пост)
/feedback - написать разработчикам
/donate - поблагодарить разработчиков материально

Также, если у вас есть желание поучавствовать в разработке бота, напишите разработчикам (команда /feedback)
